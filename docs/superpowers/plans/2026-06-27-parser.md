**Goal:** Build `apps/parser` — a standalone, independently-runnable step that parses a decompiled Java `sources/` tree once into a reusable AST + symbol index (`AstIndex`) and persists it as an `ast-index/` artifact, exposing FlowDroid-style resolved method signatures to the downstream `taint` engine.

**Architecture:** A Gradle module exposing one public library class `AstIndex` (consumed in-process by `apps/taint`) plus a picocli `parse` CLI wrapper. `AstIndex.build()` configures JavaParser with a `CombinedTypeSolver` (`ReflectionTypeSolver` + `JavaParserTypeSolver` over the source root) and a `JavaSymbolSolver`, then parses every `.java` file fail-soft: unparseable files are skipped, unresolvable types raise catchable exceptions that are swallowed so analysis continues. Native serialization of JavaParser ASTs is impractical, so `save()` persists only a small JSON descriptor of the source set + solver config and `load()` **re-parses** from it (parse-once-per-run; load re-parses).

**Tech Stack:** Java 17; Gradle (Groovy DSL); JUnit 5 + AssertJ; picocli 4.7.x; Jackson from the root `jacksonVersion`; `com.github.javaparser:javaparser-symbol-solver-core:3.26.2`.

```java
package com.oversecured.sast.parser;

import com.github.javaparser.ParserConfiguration;
import com.github.javaparser.ast.CompilationUnit;
import com.github.javaparser.ast.expr.MethodCallExpr;
import com.github.javaparser.symbolsolver.JavaSymbolSolver;
import com.github.javaparser.symbolsolver.resolution.typesolvers.CombinedTypeSolver;
import com.github.javaparser.symbolsolver.resolution.typesolvers.JavaParserTypeSolver;
import com.github.javaparser.symbolsolver.resolution.typesolvers.ReflectionTypeSolver;
import com.github.javaparser.utils.SourceRoot;
import java.io.UncheckedIOException;
import java.nio.file.Path;
import java.util.List;
import java.util.Optional;

/** Parse-once AST + symbol index over a decompiled Java source tree. */
public final class AstIndex {

    private final Path sourcesDir;
    private final List<CompilationUnit> units;

    private AstIndex(Path sourcesDir, List<CompilationUnit> units) {
        this.sourcesDir = sourcesDir;
        this.units = List.copyOf(units);
    }

    /**
     * Parse every {@code .java} file under {@code sourcesDir} with symbol resolution.
     * Fail-soft (noclasspath-style): unparseable files are skipped and unresolved
     * types never abort the build.
     */
    public static AstIndex build(Path sourcesDir) {
        Path root = sourcesDir.toAbsolutePath().normalize();

        CombinedTypeSolver typeSolver = new CombinedTypeSolver();
        typeSolver.add(new ReflectionTypeSolver());
        typeSolver.add(new JavaParserTypeSolver(root));

        ParserConfiguration config = new ParserConfiguration()
            .setLanguageLevel(ParserConfiguration.LanguageLevel.JAVA_17)
            .setSymbolResolver(new JavaSymbolSolver(typeSolver));

        SourceRoot sourceRoot = new SourceRoot(root, config);
        try {
            // tryToParse collects results; unsuccessful (broken) files are not added
            // to getCompilationUnits(), so they are silently skipped.
            sourceRoot.tryToParse();
        } catch (Exception e) {
            // Never abort the whole index because of I/O or a pathological file.
            // Fall through with whatever was parsed.
        }
        return new AstIndex(root, sourceRoot.getCompilationUnits());
    }

    public List<CompilationUnit> units() {
        return units;
    }

    // build()/save()/load()/resolveSignature() completed in later tasks.
    Path sourcesDir() {
        return sourcesDir;
    }

    public void save(Path indexDir) {
        throw new UnsupportedOperationException("implemented in Task 4");
    }

    public static AstIndex load(Path indexDir) {
        throw new UnsupportedOperationException("implemented in Task 4");
    }

    public Optional<String> resolveSignature(MethodCallExpr call) {
        throw new UnsupportedOperationException("implemented in Task 3");
    }
}
```

```java
    public Optional<String> resolveSignature(MethodCallExpr call) {
        try {
            ResolvedMethodDeclaration decl = call.resolve();
            return Optional.of(toFlowDroidSignature(decl));
        } catch (RuntimeException e) {
            // UnsolvedSymbolException and any resolver failure -> fail-soft.
            return Optional.empty();
        }
    }

    private static String toFlowDroidSignature(ResolvedMethodDeclaration decl) {
        StringBuilder params = new StringBuilder();
        for (int i = 0; i < decl.getNumberOfParams(); i++) {
            if (i > 0) {
                params.append(',');
            }
            params.append(decl.getParam(i).getType().describe());
        }
        return "<" + decl.declaringType().getQualifiedName()
            + ": " + decl.getReturnType().describe()
            + " " + decl.getName()
            + "(" + params + ")>";
    }
```

**Interfaces:**
- Consumes: `AstIndex.build` + `sourcesDir()` (Task 2).
- Produces:
  - `record IndexMeta(int version, String sourcesDir, String languageLevel)` — Jackson-serializable descriptor.
  - `void save(java.nio.file.Path indexDir)` — writes `indexDir/index-meta.json`.
  - `static AstIndex load(java.nio.file.Path indexDir)` — reads the descriptor and **re-parses** via `build(sourcesDir)`.

**Design note (document verbatim in the class Javadoc):** JavaParser ASTs hold non-serializable resolver state and parent links; persisting them natively is impractical. So the `ast-index/` artifact stores only a JSON descriptor of the source set + solver config; `load()` re-parses the sources. Tradeoff: **parse-once-per-run** (build → save within a run is cheap to reproduce), **load re-parses** the original `sources/`. The original `sources/` artifact must remain available to the downstream `taint` step (it is, per spec §3.5 artifact store).

```java
    private static final ObjectMapper MAPPER = new ObjectMapper();

    /**
     * Persist the index. JavaParser ASTs are not natively serializable, so this
     * writes only a JSON descriptor of the source set + solver config;
     * {@link #load(Path)} re-parses from it (parse-once-per-run; load re-parses).
     */
    public void save(Path indexDir) {
        try {
            Files.createDirectories(indexDir);
            IndexMeta meta = new IndexMeta(
                IndexMeta.CURRENT_VERSION,
                sourcesDir.toString(),
                ParserConfiguration.LanguageLevel.JAVA_17.name());
            MAPPER.writerWithDefaultPrettyPrinter()
                .writeValue(indexDir.resolve("index-meta.json").toFile(), meta);
        } catch (IOException e) {
            throw new UncheckedIOException("failed to write ast-index to " + indexDir, e);
        }
    }

    /** Re-parse the sources described by {@code indexDir/index-meta.json}. */
    public static AstIndex load(Path indexDir) {
        try {
            IndexMeta meta = MAPPER.readValue(
                indexDir.resolve("index-meta.json").toFile(), IndexMeta.class);
            return build(Path.of(meta.sourcesDir()));
        } catch (IOException e) {
            throw new UncheckedIOException("failed to read ast-index from " + indexDir, e);
        }
    }
```

```java
package com.oversecured.sast.parser.cli;

import com.oversecured.sast.parser.AstIndex;
import java.nio.file.Path;
import java.util.concurrent.Callable;
import picocli.CommandLine;
import picocli.CommandLine.Command;
import picocli.CommandLine.Option;

@Command(
    name = "parse",
    description = "Parse a decompiled Java source tree into a reusable ast-index artifact.")
public final class ParseCommand implements Callable<Integer> {

    @Option(names = "--src", required = true, description = "Decompiled sources/ directory.")
    Path src;

    @Option(names = "--out", required = true, description = "Output ast-index/ directory.")
    Path out;

    @Override
    public Integer call() {
        AstIndex index = AstIndex.build(src);
        index.save(out);
        System.out.println("Parsed " + index.units().size()
            + " compilation unit(s); ast-index written to " + out.toAbsolutePath());
        return 0;
    }

    public static int run(String... args) {
        return new CommandLine(new ParseCommand()).execute(args);
    }

    public static void main(String[] args) {
        System.exit(run(args));
    }
}
```

```java
import org.junit.jupiter.api.io.TempDir;

            .as("fail-soft parse still yields compilation units")
            .isNotEmpty();

        assertThat(reloaded.units())
            .anySatisfy(cu ->
                assertThat(cu.getPrimaryTypeName()).hasValue("WebViewActivity"));

    @Test
    void resolvesDeterministicFixtureSignatureWithoutAndroidJar() {
        Path fixture = Paths.get("src", "test", "resources", "fixtures", "simple");

        AstIndex index = AstIndex.build(fixture);
        MethodCallExpr loadUrl = index.units().stream()
            .flatMap(cu -> cu.findAll(MethodCallExpr.class).stream())
            .filter(c -> c.getNameAsString().equals("loadUrl"))
            .findFirst()
            .orElseThrow();

        assertThat(sig).isPresent();
        assertThat(sig.get())
            .contains("com.example.web.WebView")

        assertThat(Files.isDirectory(src)).isTrue();

        AstIndex built = AstIndex.build(src);
        built.save(out);
        AstIndex loaded = AstIndex.load(out);

        assertThat(loaded.units()).hasSameSizeAs(built.units());
    }
```

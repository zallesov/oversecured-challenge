**Shared conventions:** [Shared Contracts and Naming Conventions](../reference/2026-06-27-shared-contracts-and-conventions.md)
**Logging:** [Logging Conventions](../reference/2026-06-27-logging.md)
**Error handling:** [Error Handling Conventions](../reference/2026-06-27-error-handling.md)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the Gradle multi-module root and the `common` library that defines every shared contract (findings schema, manifest facts, artifact store, signature parser, JSON util) consumed by all other modules in the Android Taint SAST Pipeline.

**Architecture:** A Gradle multi-module Groovy build whose root `build.gradle` pins Java 17, JUnit 5, AssertJ, and Jackson versions in one `ext` block applied to every subproject. The `common` module holds only stable cross-cutting types — Java 17 records and interfaces — plus a single shared Jackson `ObjectMapper` for (de)serializing the `findings.json` and `facts.json` schemas. No analysis logic lives here; `common` is the dependency of every step app and the orchestrator.

**Tech Stack:** Java 17 (toolchain), Gradle (Groovy DSL, multi-module), JUnit 5 (Jupiter) + AssertJ, Jackson (`jackson-databind` + `jackson-dataformat-yaml`).

## Global Constraints

- Java 17 — every module uses `java { toolchain { languageVersion = JavaLanguageVersion.of(17) } }`. (Verbatim floor from PROJECT TECH STACK: "Java 17".)
- Gradle multi-module, **Groovy** `build.gradle` files (not Kotlin DSL).
- Root `settings.gradle` includes **all** modules: `common`, `apps:decompiler`, `apps:parser`, `apps:manifest-facts`, `apps:taint`, `apps:manifest-misconfig`, `apps:reporter`, `orchestrator`, `benchmark`.
- Dependency versions declared **once** in the root `ext` block; subprojects reference `rootProject.ext.*`. No version literals scattered across module builds.
- Tests: JUnit 5 (Jupiter) + AssertJ only. Each module build calls `test { useJUnitPlatform() }`.
- JSON/YAML serialization: Jackson (`jackson-databind` + `jackson-dataformat-yaml`). One shared `ObjectMapper`.
- Runnable apps use the Gradle `application` plugin (apps/* and orchestrator); `common` and `benchmark` are libraries (`java-library`), not applications.
- Signature grammar (spec §5): FlowDroid/Soot-style `<fully.qualified.Class: ReturnType methodName(ParamType,ParamType,...)>`; inner classes use `$`.
- `findings.json` is the analyzer-agnostic shared schema (spec §7); `facts.json` is the manifest-facts schema (spec §3.4). Both are defined here and nowhere else.
- `S3Store` is a **documented future stub** (spec §3.5): no implementation in this plan, only a Javadoc note where `LocalFsStore` lives.
- All commits in this plan use the conventional-commit style shown in each Step 5.

```java
package com.oversecured.sast.common;

import com.fasterxml.jackson.databind.DeserializationFeature;
import com.fasterxml.jackson.databind.ObjectMapper;
import com.fasterxml.jackson.databind.SerializationFeature;

/** Shared Jackson ObjectMapper and read/write helpers for the common schemas. */
public final class Json {

    private static final ObjectMapper MAPPER = new ObjectMapper()
            .enable(SerializationFeature.INDENT_OUTPUT)
            .disable(DeserializationFeature.FAIL_ON_UNKNOWN_PROPERTIES);

    private Json() {
    }

    public static ObjectMapper mapper() {
        return MAPPER;
    }

    public static byte[] writeBytes(Object value) {
        try {
            return MAPPER.writeValueAsBytes(value);
        } catch (Exception e) {
            throw new RuntimeException("JSON serialization failed", e);
        }
    }

    public static String writeString(Object value) {
        try {
            return MAPPER.writeValueAsString(value);
        } catch (Exception e) {
            throw new RuntimeException("JSON serialization failed", e);
        }
    }

    public static <T> T read(byte[] data, Class<T> type) {
        try {
            return MAPPER.readValue(data, type);
        } catch (Exception e) {
            throw new RuntimeException("JSON deserialization failed", e);
        }
    }
}
```

```java
package com.oversecured.sast.common;

import java.util.List;
import java.util.regex.Matcher;
import java.util.regex.Pattern;

/**
 * Parses FlowDroid/Soot-style signatures: {@code <fqcn: ret name(p,p,...)>}.
 * Inner classes use '$'; the method name may be {@code <init>} or {@code <clinit>}.
 */
public final class SignatureParser {

    // <  class  :  ret  name  ( params )  >
    private static final Pattern SIG = Pattern.compile(
            "^<([^:]+):\\s+(\\S+)\\s+(<init>|<clinit>|[\\w$]+)\\((.*)\\)>$");

    private SignatureParser() {
    }

    public static MethodSignature parse(String sig) {
        if (sig == null) {
            throw new IllegalArgumentException("signature is null");
        }
        Matcher m = SIG.matcher(sig.trim());
        if (!m.matches()) {
            throw new IllegalArgumentException("malformed signature: " + sig);
        }
        String declaringClass = m.group(1).trim();
        String returnType = m.group(2).trim();
        String name = m.group(3).trim();
        String paramBlob = m.group(4).trim();

        List<String> params = paramBlob.isEmpty()
                ? List.of()
                : List.of(paramBlob.split("\\s*,\\s*"));

        return new MethodSignature(declaringClass, returnType, name, params);
    }
}
```

```java
package com.oversecured.sast.common;

import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;

/**
 * Filesystem-backed ArtifactStore — the challenge default (shared volume / local dir).
 * Keys may contain '/' to form nested paths under {@code root}.
 *
 * <p>Future work: an {@code S3Store} implementing {@link ArtifactStore} backs the cloud
 * deployment (MinIO locally, S3 in prod). It is intentionally not implemented here;
 * steps depend only on {@link ArtifactStore}, so swapping in S3Store requires no step
 * code changes (spec §3.5).
 */
public final class LocalFsStore implements ArtifactStore {

    private final Path root;

    public LocalFsStore(Path root) {
        this.root = root;
    }

    @Override
    public void put(String key, byte[] data) {
        Path target = root.resolve(key);
        try {
            if (target.getParent() != null) {
                Files.createDirectories(target.getParent());
            }
            Files.write(target, data);
        } catch (IOException e) {
            throw new RuntimeException("failed to put artifact: " + key, e);
        }
    }

    @Override
    public byte[] get(String key) {
        Path target = root.resolve(key);
        try {
            return Files.readAllBytes(target);
        } catch (IOException e) {
            throw new RuntimeException("failed to get artifact: " + key, e);
        }
    }
}
```

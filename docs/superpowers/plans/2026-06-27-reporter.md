# Reporter (Step 6, fan-in) Implementation Plan

**Shared conventions:** [Shared Contracts and Naming Conventions](../reference/2026-06-27-shared-contracts-and-conventions.md)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `apps/reporter` — the fan-in CLI that merges every analyzer's `findings.json` into one human-readable HTML report and one machine-readable SARIF v2.1.0 report.

**Architecture:** A Gradle module that depends on `common` for the shared finding model. Three pure functions compose: `FindingsMerger` (read N JSON files → `List<Finding>`), `HtmlReportRenderer` (`List<Finding>` → HTML string), `SarifReportWriter` (`List<Finding>` → SARIF JSON string). A thin picocli `ReportCommand` wires file I/O around them. Both outputs are derived from the same merged `List<Finding>` so they never diverge.

**Tech Stack:** Java 17; Gradle (Groovy DSL); JUnit 5 + AssertJ; picocli 4.7.x; Jackson from the root `jacksonVersion`. HTML is rendered by a small hand-rolled `StringBuilder` template (no template engine) — justified below.

## Global Constraints

- **Language/JDK:** Java 17 (spec §2: "modern Java"). `sourceCompatibility = 17`.
- **Build:** Gradle, Groovy DSL (`build.gradle`, not `.kts`).
- **Tests:** JUnit 5 (`org.junit.jupiter`) + AssertJ (`org.assertj.core.api.Assertions.assertThat`).
- **CLI:** picocli `info.picocli:picocli:4.7.6`.
- **JSON:** Jackson `com.fasterxml.jackson.core:jackson-databind:${rootProject.ext.jacksonVersion}` (records supported natively since 2.12 — no extra parameter-names module needed).
- **Module path:** all code under `apps/reporter`; shared contracts under `common` (CONSUME — see "Shared Contracts" below).
- **SARIF:** version string MUST be exactly `"2.1.0"` (spec §7.3).
- **Reporter is analyzer-agnostic:** it depends ONLY on the shared `Finding`/`FindingsDoc` model, never on any analyzer internals (spec §3.6, README).
- **Empty input MUST NOT crash:** zero findings → a valid empty HTML report and a valid SARIF doc with 1 run and 0 results (README "Tests").

### HTML approach decision (justification required by brief)

**Chosen: a hand-rolled `StringBuilder` template plus a dedicated `HtmlEscaper`.** Justification: the report is a single static page with a fixed shape (group-by-severity → finding cards → ordered flow list). A template engine (Thymeleaf/Freemarker/Mustache) adds a dependency, a template-resolution lifecycle, and a second file format to test, for zero benefit at this size. A hand-rolled renderer is one self-contained class, has no runtime deps, produces deterministic output that is trivial to assert with `contains(...)`, and the only real risk — HTML injection from finding text — is isolated in one tested `HtmlEscaper`. This is the simplest thing that works (YAGNI).

The `common` foundation plan owns the canonical all-module root build and the shared finding records in package `com.oversecured.sast.common`. This reporter plan consumes those records; it must not create a parallel shared-model package or overwrite the root build with a reporter-only module set.

```java
package com.oversecured.sast.reporter;

import com.fasterxml.jackson.databind.DeserializationFeature;
import com.fasterxml.jackson.databind.ObjectMapper;
import com.oversecured.sast.common.Finding;
import com.oversecured.sast.common.FindingsDoc;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.ArrayList;
import java.util.List;

/** Reads multiple analyzer findings.json files and concatenates their findings. */
public final class FindingsMerger {

  private static final ObjectMapper MAPPER =
      new ObjectMapper().configure(DeserializationFeature.FAIL_ON_UNKNOWN_PROPERTIES, false);

  public static ObjectMapper mapper() {
    return MAPPER;
  }

  public List<Finding> merge(List<Path> findingsFiles) throws IOException {
    List<Finding> merged = new ArrayList<>();
    for (Path file : findingsFiles) {
      FindingsDoc doc = MAPPER.readValue(Files.readAllBytes(file), FindingsDoc.class);
      if (doc.findings() != null) {
        merged.addAll(doc.findings());
      }
    }
    return merged;
  }
}
```

```java
package com.oversecured.sast.reporter;

/** Minimal HTML-text escaper for untrusted finding strings. */
public final class HtmlEscaper {

  private HtmlEscaper() {}

  public static String escape(String s) {
    if (s == null) {
      return "";
    }
    StringBuilder out = new StringBuilder(s.length() + 16);
    for (int i = 0; i < s.length(); i++) {
      char c = s.charAt(i);
      switch (c) {
        case '&' -> out.append("&amp;");
        case '<' -> out.append("&lt;");
        case '>' -> out.append("&gt;");
        case '"' -> out.append("&quot;");
        case '\'' -> out.append("&#39;");
        default -> out.append(c);
      }
    }
    return out.toString();
  }
}
```

`apps/reporter/src/main/java/com/oversecured/sast/reporter/HtmlReportRenderer.java`:
```java
package com.oversecured.sast.reporter;

import static com.oversecured.sast.reporter.HtmlEscaper.escape;

import com.oversecured.sast.common.Finding;
import com.oversecured.sast.common.FlowStep;
import com.oversecured.sast.common.Severity;
import java.util.List;

/** Renders merged findings into a single static HTML document (hand-rolled template). */
public final class HtmlReportRenderer {

  public String render(List<Finding> findings) {
    StringBuilder b = new StringBuilder();
    b.append("<!DOCTYPE html>\n");
    b.append("<html lang=\"en\"><head><meta charset=\"utf-8\">");
    b.append("<title>Android Taint SAST Report</title>");
    b.append("<style>")
        .append("body{font-family:system-ui,sans-serif;margin:2rem;}")
        .append(".finding{border:1px solid #ccc;border-radius:6px;padding:1rem;margin:1rem 0;}")
        .append(".sev-ERROR{border-left:6px solid #c0392b;}")
        .append(".sev-WARNING{border-left:6px solid #d68910;}")
        .append(".sev-NOTE{border-left:6px solid #2980b9;}")
        .append("code{background:#f4f4f4;padding:0 .25rem;}")
        .append("</style></head><body>");
    b.append("<h1>Android Taint SAST Report</h1>");

    if (findings.isEmpty()) {
      b.append("<p>No findings.</p>");
    } else {
      for (Severity severity : Severity.values()) {
        List<Finding> group = findings.stream().filter(f -> f.severity() == severity).toList();
        if (group.isEmpty()) {
          continue;
        }
        b.append("<section><h2>").append(severity.name())
            .append(" (").append(group.size()).append(")</h2>");
        for (Finding f : group) {
          renderFinding(b, f);
        }
        b.append("</section>");
      }
    }

    b.append("</body></html>");
    return b.toString();
  }

  private void renderFinding(StringBuilder b, Finding f) {
    b.append("<div class=\"finding sev-").append(escape(f.severity().name())).append("\">");
    b.append("<h3>").append(escape(f.ruleId())).append("</h3>");
    b.append("<p>").append(escape(f.message())).append("</p>");
    b.append("<p><strong>Class:</strong> ").append(escape(f.vulnerabilityClass()))
        .append(" &middot; <strong>").append(escape(f.cwe())).append("</strong>")
        .append(" &middot; OWASP ").append(escape(f.owaspMobile())).append("</p>");

      }
      b.append("</ol>");
    }

      b.append("<h4>Notes</h4><ul>");
      for (String note : f.notes()) {
        b.append("<li>").append(escape(note)).append("</li>");
      }
      b.append("</ul>");
    }

    b.append("</div>");
  }
}
```

```java
package com.oversecured.sast.reporter;

import com.fasterxml.jackson.core.JsonProcessingException;
import com.fasterxml.jackson.databind.ObjectMapper;
import com.fasterxml.jackson.databind.node.ArrayNode;
import com.fasterxml.jackson.databind.node.ObjectNode;
import com.oversecured.sast.common.Finding;
import com.oversecured.sast.common.FlowStep;
import com.oversecured.sast.common.Severity;
import java.util.ArrayList;
import java.util.LinkedHashMap;
import java.util.List;
import java.util.Map;

/** Builds a SARIF v2.1.0 document from merged findings. */
public final class SarifReportWriter {

  private static final ObjectMapper MAPPER = FindingsMerger.mapper();
  private static final String SCHEMA = "https://json.schemastore.org/sarif-2.1.0.json";

  public ObjectNode toSarif(List<Finding> findings) {
    // First-seen-ordered rule registry → ruleIndex.
    Map<String, Finding> ruleById = new LinkedHashMap<>();
    for (Finding f : findings) {
      ruleById.putIfAbsent(f.ruleId(), f);
    }
    List<String> ruleOrder = new ArrayList<>(ruleById.keySet());

    ObjectNode root = MAPPER.createObjectNode();
    root.put("version", "2.1.0");
    root.put("$schema", SCHEMA);

    ArrayNode runs = root.putArray("runs");
    ObjectNode run = runs.addObject();

    ObjectNode driver = run.putObject("tool").putObject("driver");
    driver.put("name", "android-taint-sast");
    driver.put("version", "0.1.0");
    driver.put("informationUri", "https://github.com/oversecured/challenge");
    ArrayNode rules = driver.putArray("rules");
    for (String ruleId : ruleOrder) {
      Finding f = ruleById.get(ruleId);
      ObjectNode rule = rules.addObject();
      rule.put("id", f.ruleId());
      rule.put("name", f.vulnerabilityClass());
      rule.putObject("shortDescription").put("text", f.message());
      rule.putObject("defaultConfiguration").put("level", level(f.severity()));
      ObjectNode props = rule.putObject("properties");
      props.put("cwe", f.cwe());
      props.put("owaspMobile", f.owaspMobile());
      props.put("vulnerabilityClass", f.vulnerabilityClass());
    }

    ArrayNode results = run.putArray("results");
    for (Finding f : findings) {
      ObjectNode result = results.addObject();
      result.put("ruleId", f.ruleId());
      result.put("ruleIndex", ruleOrder.indexOf(f.ruleId()));
      result.put("level", level(f.severity()));
      result.putObject("message").put("text", f.message());

      if (f.flow() != null && !f.flow().isEmpty()) {
        FlowStep sink = f.flow().get(f.flow().size() - 1);
        result.putArray("locations").add(physicalLocationWrapper(sink));

        ArrayNode tfLocations = result.putArray("codeFlows")
            .addObject().putArray("threadFlows")
            .addObject().putArray("locations");
        for (FlowStep step : f.flow()) {
          ObjectNode loc = tfLocations.addObject();
          ObjectNode location = loc.putObject("location");
          addPhysicalLocation(location, step);
          location.putObject("message").put("text", step.label());
        }
      }
    }

    return root;
  }

  public String toSarifJson(List<Finding> findings) throws JsonProcessingException {
    return MAPPER.writerWithDefaultPrettyPrinter().writeValueAsString(toSarif(findings));
  }

  private ObjectNode physicalLocationWrapper(FlowStep step) {
    ObjectNode wrapper = MAPPER.createObjectNode();
    addPhysicalLocation(wrapper, step);
    return wrapper;
  }

  private void addPhysicalLocation(ObjectNode parent, FlowStep step) {
    ObjectNode physical = parent.putObject("physicalLocation");
    physical.putObject("artifactLocation").put("uri", step.file());
    physical.putObject("region").put("startLine", step.line());
  }

  private static String level(Severity severity) {
    return switch (severity) {
      case ERROR -> "error";
      case WARNING -> "warning";
      case NOTE -> "note";
    };
  }
}
```

```java
package com.oversecured.sast.reporter;

import com.oversecured.sast.common.Finding;
import java.nio.charset.StandardCharsets;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.ArrayList;
import java.util.List;
import java.util.concurrent.Callable;
import picocli.CommandLine;
import picocli.CommandLine.Command;
import picocli.CommandLine.Option;

@Command(
    name = "report",
    mixinStandardHelpOptions = true,
    description = "Merge analyzer findings.json files into HTML + SARIF reports.")
public final class ReportCommand implements Callable<Integer> {

  @Option(
      names = "--findings",
      arity = "1..*",
      description = "Findings JSON files to merge (steps 4 & 5).")
  List<Path> findings = new ArrayList<>();

  @Option(names = "--out-html", required = true, description = "HTML report output path.")
  Path outHtml;

  @Option(names = "--out-sarif", required = true, description = "SARIF v2.1.0 output path.")
  Path outSarif;

  @Override
  public Integer call() throws Exception {
    List<Finding> merged = new FindingsMerger().merge(findings);

    String html = new HtmlReportRenderer().render(merged);
    String sarif = new SarifReportWriter().toSarifJson(merged);

    writeFile(outHtml, html);
    writeFile(outSarif, sarif);
    return 0;
  }

  private static void writeFile(Path path, String content) throws Exception {
    if (path.getParent() != null) {
      Files.createDirectories(path.getParent());
    }
    Files.write(path, content.getBytes(StandardCharsets.UTF_8));
  }

  public static void main(String[] args) {
    System.exit(new CommandLine(new ReportCommand()).execute(args));
  }
}
```
